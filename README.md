
## Гиперпараметры: 

Значения:
- **alpha** = 0.5, 
- **beta** = 5.0, 
- **phi** = 0.3, 
- **q** = 100, 
- **num_ants** = 10, 
- **iterations** = 100

Обозначения:

- **alpha** = влияние феромонов, 
- **beta** = влияние эвристики, 
- **phi** = коэффициент испарения феромонов, 
- **q** = количество феромонов для добавления, 
- **num_ants** = количество муравьев, 
- **iterations** = количество итераций
